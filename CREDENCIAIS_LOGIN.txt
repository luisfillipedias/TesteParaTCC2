  RegulaSUS — Credenciais de Acesso (Teste)
  Login via gov.br (Simulação)
============================================

O login é feito pelo CPF + Senha, simulando o Login Único gov.br.
Acesse: http://localhost:5173/login (ou a URL da Vercel)

--------------------------------------------

COMO USAR:
1. Acesse a página de login
2. Digite o CPF (só números ou formatado, tanto faz)
3. Clique em "Avançar"
4. Digite a senha correspondente
5. Clique em "Entrar"
6. Você será redirecionado para o dashboard do perfil

DASHBOARDS POR PERFIL:
- Médico   → /medico/dashboard
- Paciente → /paciente/dashboard
- Gestor   → /gestor/dashboard
- Admin    → /admin/usuarios

OBSERVAÇÕES:
- As senhas seguem padrão gov.br (maiúscula, especial, número)
- A sessão expira em 1 hora
- Para sair, clique em "Sair do sistema" na sidebar
- Acessar um dashboard sem login redireciona para o login
