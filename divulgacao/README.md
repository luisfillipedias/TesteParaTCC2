# Divulgacao

Esta pasta e destinada a materiais de divulgacao do projeto.

## Conteudo sugerido

- `apresentacoes/`: slides de seminarios e bancas.
- `videos/`: demonstracoes, pitch e apresentacoes gravadas.
- posters, cards e outros materiais de comunicacao.

## Observacao

Evite publicar arquivos com dados pessoais sem consentimento.
